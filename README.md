# 💳 Credit Card Approval Prediction

## 📌 Project Overview

The **Credit Card Approval Prediction** project is a Machine Learning web application that predicts whether a credit card application is likely to be approved based on user inputs such as **Age**, **Annual Income**, and **Credit Score**.

The application provides a simple and user-friendly interface where users enter their details and receive an instant prediction.

**Live Appilication:**https://credit-card-approval-prediction18.onrender.com
**Demo video link:**https://youtu.be/tmcsUhMR908?si=qBEfYujzoVO2_as6

## 🎯 Objectives

- Predict credit card approval using Machine Learning.
- Reduce manual verification time.
- Provide quick and accurate approval predictions.
- Demonstrate the practical application of Machine Learning in banking.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Mallika18041/credit-card-approval-prediction18.git
```

Move into the project folder

```bash
cd credit-card-approval-prediction18
```

Install required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000/
```
